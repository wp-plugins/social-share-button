== Description ==

Display social share buttons under post or page.

easy to use via admin settings to control different style for social share buttons

N.B. Initial version not able to display share count stats.
we are updating



== Installation ==


1. Install as regular WordPress plugin.
2. Go your Pluings setting via WordPress Dashboard and <strong>activate</strong> it.
3. After Activate you see Menu on left Sidebar "<strong>SSB Settings</strong>" 

if you are first time chage settigns and hit <strong>"Save Changes"</strong>
